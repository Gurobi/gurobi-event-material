# Gurobi event material
Public material (code, exercises and more) for Gurobi events

## Instructions

- This repository should only contain public information
- Presentations should be stored in https://github.com/gurobi/slides
- Material is actively being used during Gurobi events. Each folder should be treated with care, since links to the files may be distributed before, during and after events through e-mail, QR codes and more. Don't rename files without permission from the event organizer.
